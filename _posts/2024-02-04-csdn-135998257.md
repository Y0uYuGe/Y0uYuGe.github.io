# 有车阻挡
先踩刹车或者自然降速，估计要停了就踩离合换1档，然后踩刹车，再半制动起步，起步后再踩油门换2挡，然后维持速度。如果估计不会停，就等距离合适后再踩油门启动。（一般是停好些）
# 准备
上车，系安全带，调座位和镜子，检测灯光，手刹，挡位，远近交替灯闪两下，然后踩离合和刹车，转钥匙，一转到低，然后等待灯光考试

# 灯光
超车，弯道，拱桥，人行道，坡道：远近交替
照明不良：远光灯
路边停车：小灯
跟车，会车，有信号，照明良好：近光灯

# 起步
打左转向灯，踩刹车和离合，换1挡，拉手刹，松刹车，慢慢松完离合后，把左脚放到一边，踩油门，到15的时候，踩离合，换2档，松一半离合，再松一半离合，踩油门，稳定到2档

# 加减档
踩油门到25，踩死离合，换3档，踩油门到35，踩离合，换4档，踩油门维持到40-45大概5秒后，踩刹车，慢慢减速到25，松刹车，然后踩离合，换2档，松一半，再松一半，然后继续踩油门

# 调头
打左转向灯，松油门，让汽车行驶一段距离并慢慢降速，速度到10后，踩离合，换1档，同时脚踩刹车，使其速速度再降一些，速度较慢后，松刹车，慢慢松离合，同时脚也看机会踩刹车降速，等车过来斑马线后，方向盘向左打死，等车回正后，直接松离合，踩油门，到15时候，松油门，踩离合，换2档，离合松一半，再松一半，踩油门，稳定速度在2档

# 侧边停车
打开右转向灯，踩刹车，降速到10，松刹车，踩死离合，换1档，不松离合，再次踩刹车控制速度， 同时看雨刮器右边刚碰着右边边线即可，当碰着一段时间后踩刹车使得车停，始终保持右转向灯开启，关闭就再开，先回正，再靠边上

# 下车
保持右转向灯，,挂空档，拉手刹，熄火，松刹车，松离合，观察后视镜，开门一下看后面，再关门
# 超车
打开左转向灯，数三秒后，观察后视镜，踩油门加速左转，回正后，打开右转向灯，三秒后，踩油门加速右转，回正后关闭右转向灯，继续2档行驶
# 直线行驶
雨刮器右边于右边线的最左边挨着，当指令还在播报时候，去调整雨刮器与右边线的位置，直到方向盘正了之后，这个距离也不会变化，指令播报结束后，微调方向盘小于5度保证雨刮器与右边线的位置不变，同时踩油门到35后松开油门，然后降到25再继续踩着油门维持
# 左转
先打左转向灯，变道，松油门，让汽车自然降速，到10时候，踩离合，换1档，此时如果仍需停车，就踩刹车，绿灯后，半制动，起步后完全松开离合然后开始踩油门，稳定到速度依然是20以下，等车出斑马线后向左打半圈，等车进入路后，回正方向盘，加速到20，踩离合，换2档，继续行驶
# 右转
与左转类似，只不过转弯路口不一样
# 公交站
车身进入公交站后踩下刹车
# 人行道
车头离人行道还有一段距离时踩刹车一下
