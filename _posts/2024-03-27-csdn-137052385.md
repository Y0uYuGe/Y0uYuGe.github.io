# mount

设备存储结构就是一个文件系统，挂载就是对应到电脑中就是关联到一个目录结构。即映射
卸载就是将这个映射解除同时将修改生效到原来的设备存储内容中去

mount命令是在Linux和类Unix系统中用于将文件系统挂载到指定的挂载点（目录）的命令。通过挂载，可以使得文件系统中的文件和目录在指定的挂载点下可访问。

mount命令的基本语法为：

```bash
mount [options] <device> <mount_point>
```

 
其中，<device> 参数指定要挂载的设备或分区，可以是硬盘、分区、磁盘镜像等等。<mount_point> 参数指定将设备挂载到的目录。

mount命令的一些常见选项包括： - -t <type>：指定文件系统的类型，可以是ext4、ntfs、xfs等等。 - -o <options>：指定挂载选项，例如读写权限、权限控制等。 - -r：以只读方式挂载设备，禁止写入操作。 - -o remount：重新挂载已经挂载的设备，可以更改挂载选项。

例如，如果要将/dev/sdb1设备挂载到/mnt目录下，可以使用以下命令：

```bash
mount /dev/sdb1 /mnt
```

 
通过mount命令可以查看当前系统中已挂载的文件系统信息，使用mount命令即可显示当前挂载点和设备的对应关系。

# cgroup文件系统中的相关组件

- cgroups：就是文件夹，其中包含一些当前cgroup的配置文件（以文件形式）和子cgroups的文件夹
- subsystem：和系统相关，负责管理不同操作系统功能，再进一步来说功能对应到的进程，再进一步说对应到管理的进程使用的资源。对应到cgroups中的每个cgroups的进程的资源限制（以文件形式规定）
- hierarchy：由cgoup组成的类似树的组织目录结构

# docker 使用cgroups
docker run一个镜像是启动了一个容器，会给容器分配一个cgroup

> 在 Ubuntu 22.04 中，Docker 容器的 cgroup 位于 /sys/fs/cgroup 目录下的 system.slice 目录中。每个容器会在 system.slice 目录下创建一个对应的目录，目录名称类似于 docker-<container_id>.scope

# go代码

- /proc/self/exe是一个特殊的路径，它指向当前正在运行的可执行文件。而os.Args[0]是程序的第一个命令行参数，即程序本身的路径。
- 


```go
package main

import (
	"os/exec"
	"os"
	"syscall"
	"io/ioutil"
	"fmt"
	"strconv"
	"path"
	"bufio"
)

const process_cgroup_root_path="/sys/fs/cgroup/"

func main(){  
	if os.Args[0]=="/proc/self/exe"{  //检查是否是fork出来的新go程序进程
	cmd:=exec.Command("/proc/self/exe")
	fmt.Println("args[0]",os.Args[0])
	fmt.Println("/proc/self/exe",cmd)
	fmt.Println()
	cmd=exec.Command("sh","-c","stress --vm-bytes 200m --vm-keep -m 1")
	cmd.SysProcAttr = &syscall.SysProcAttr{}
	cmd.Stdin = os.Stdin 
	cmd.Stdout = os.Stdout 
	cmd.Stderr = os.Stderr 
	bufio.NewReader(os.Stdin).ReadString('\n') //等待回车再继续往下进行，防止cgroup还没有修改完就开始执行stress了
	if err:=cmd.Run(); err!= nil{ 
	 fmt.Println(err)
	  os.Exit(1)
	}
	fmt.Println("finish")
	}

	cmd:= exec.Command ("/proc/self/exe") 	
	cmd.SysProcAttr = &syscall.SysProcAttr{ 
		Cloneflags: syscall.CLONE_NEWIPC|syscall.CLONE_NEWPID|syscall.CLONE_NEWUTS|syscall.CLONE_NEWNS|syscall.CLONE_NEWUSER|syscall.CLONE_NEWUSER|syscall.CLONE_NEWNET,}
	cmd.Stdin = os.Stdin
	cmd.Stdout = os.Stdout 
	cmd.Stderr = os.Stderr
	if err:= cmd.Start( ); err !=nil { 
		fmt.Println ("error", err) 
		os.Exit(1) 
	} else { 
		
		pid:=cmd.Process.Pid  
		fmt.Println("fork process pid ",pid)
		os.Mkdir(path.Join(process_cgroup_root_path,"cgroup-test"),0755)
		fmt.Println("success mkdir")
		ioutil.WriteFile(path.Join(process_cgroup_root_path, "cgroup-test","cgroup.procs"),[]byte(strconv.Itoa(cmd.Process.Pid)), 0755)
		ioutil.WriteFile(path.Join(process_cgroup_root_path,"cgroup-test","memory.max"),[]byte(strconv.Itoa(100 * 1024 * 1024)), 0755)  //注意往memory.max写只能写数字
		fmt.Println("change finished ")
		cmd.Process.Wait()  //等待start()结束，防止go程序结束
		}
}

```
