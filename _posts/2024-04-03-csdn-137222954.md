然后一个矩阵存储了两个状态的转移概率信息，然后还有一个初始状态概率信息

# 已知模型，求某个观测序列概率
##  暴力求解
即将所有可能都算一遍

即先求出各种隐藏序列的概率然后再求出各种隐藏序列下对应的该某个观察序列的概率
此时如果隐藏状态数有N个，那么对应的隐藏序列的可能性有T乘N的T次方 （T是初始状态的可能）

## 前向算法
利用动态规划

状态转移方程：**当前某个隐藏状态对应的观察状态的概率=前一个的各个隐藏状态的某个固定的观察状态乘前一个各个隐藏状态到当前隐藏状态的转移概率再乘当前隐藏状态到当前观测状态的概率**

此时如果隐藏状态数有N个，T个观察序列，那么得到当前时刻的隐藏状态的对应的观察状态的需要经历的时间复杂度为N的平方（前一个有N个固定某个观察状态的隐藏状态，当前有N个固定某个观察状态的隐藏状态），总的需要T次迭代，所以为T的N的平方

##  Baum-Welch算法
1. 简单概括就是先初始化一个模型参数
2. 然后求出可观测序列的概率表达式（由初始化的模型参数和下次迭代的模型参数决定）
3. 然后利用拉格朗日求其最大值对应的各个模型参数
4. 然后再次作为初始化模型参数迭代
![在这里插入图片描述](/images/csdn/2024/csdn-137222954/image-005-da64ee5927.png)


多次迭代后可以得到变化不太大的模型参数那么可认为是模型参数

## 维特比算法
类似动态规划，也有点递推

即到当前固定的观测状态对应的某个隐藏状态的最大概率=最大值（到达上一个固定的观测状态的对应的各个隐藏状态的概率乘该隐藏状态转移当前隐藏状态的概率）乘当前隐藏状态转化到当前可观测状态的概率

# HMM工具包使用

```python
import numpy as np
from hmmlearn import hmm

states = ["box 1", "box 2", "box3"]
n_states = len(states)

observations = ["red", "white"]
n_observations = len(observations)

start_probability = np.array([0.2, 0.4, 0.4])

transition_probability = np.array([
  [0.5, 0.2, 0.3],
  [0.3, 0.5, 0.2],
  [0.2, 0.3, 0.5]
])

emission_probability = np.array([
  [0.5, 0.5],
  [0.4, 0.6],
  [0.7, 0.3]
])

model = hmm.MultinomialHMM(n_components=n_states)
model.startprob_=start_probability
model.transmat_=transition_probability
model.emissionprob_=emission_probability

seen = np.array([[0,1,0]]).T
logprob, box = model.decode(seen, algorithm="viterbi") # 维特比算法求摸的盒子数
print("The ball picked:", ", ".join(map(lambda x: observations[x], seen)))  
print("The hidden box", ", ".join(map(lambda x: states[x], box)))

box2 = model.predict(seen)
print("The ball picked:", ", ".join(map(lambda x: observations[x], seen)))
print("The hidden box", ", ".join(map(lambda x: states[x], box2)))  # 求某个观测序列的概率

print (model.score(seen))


model2 = hmm.MultinomialHMM(n_components=n_states, n_iter=20, tol=0.01)
X2 = np.array([[0,1,0,1],[0,0,0,1],[1,0,1,1]])  # 求模型参数
model2.fit(X2)
print (model2.startprob_)
print (model2.transmat_)
print (model2.emissionprob_)
print (model2.score(X2))


```
