# 注意
系安全带
打一圈半的情况就只有倒车入库出去时。
两个地方打转向灯

# 预备
首先系安全带，然后车钥匙转90度，调整位置，直到向后靠，头 直视前方时差不多正好看不到车前盖。调节反光镜，最后踩死离合刹车，车钥匙再转90度，汽车启动。压手刹，换一档。松离合。

# 倒车入库
将左边的后视镜的下沿中点与左边的线对齐行驶。
当差不多屁股坐到前面的边界线时候，踩死离合，踩刹车
随后倒车时，左后视镜看到的轮胎压到线时，向右打死。然后白线看不见会半圈。继续倒，车身正了后看屁股是否正好坐倒那个标记上，出库时左一圈半，等差不多镜子出库就换倒挡，方向盘不动，回来时候可以打死一会会，然后依然是一圈半继续倒，车身正了就看屁股是不是正好坐到线上了（线在屁股的中间）

# 侧方停车
前面镜子的左下角对着左边的线走，当看不到前面的线时候看右边的后视镜，如果对角在中心就开始换挡倒，当看到第三个黑线（最左边算一个黑的）回正，然后入库，当轮子压线后再开始打方向，左边一圈半，车身正了后，打转向灯，换挡，方向盘不动，当左后轮过线后回正并向右打一圈

# 曲线行驶
前面镜子的左下角对着左边的线进入，当发现左后视镜的左边离开了左边的曲线，往左打一圈，始终保持，直到曲线方向变了后，就保持原来的一圈，直到你看通过车门看不到线再直着走，当右边的后视镜的下边看得到右边的曲线时候向右边打一圈+45度。直接走，等回正了就直行

# 直接行驶
前面镜子的左下角对着左边的线进入，可以早些开启左转向灯，等折角的线差不多在屁股正中间即可（或者两大腿分开的那个地方对着那根线）然后向右打死

# 坡道定点停车起步
雨刮器最高点的左边那个的右边差不多刚好贴着右边的线，上去的时候改为雨刮器最高点的左边那个的左边差不多刚好贴着右边的线，当线消失时，当第三个线出现在左后视镜下面一些时停车。拉手刹，松手刹，慢慢松离合直到车子颤动了，就送刹车，同时放宽些离合，使得向前上去
