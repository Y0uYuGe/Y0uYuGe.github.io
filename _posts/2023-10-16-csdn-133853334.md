# 百分百解决vscode终端乱码问题
## 方法1：修改注册表编辑器
**本人安装了code runner后并选择在run in terminal后发现终端乱码先是参考网上的修改注册表编辑器的方法发现cmd修改成功，但终端编码仍然是gbk，各位可以去试一试，说不定是我的电脑问题**

## 方法2：修改 terminal.integrated.profiles.windows

***打开vscode的设置搜索 terminal.integrated.profiles.windows***
![在这里插入图片描述](/images/csdn/2023/csdn-133853334/image-003-0cd3676fee.png)
***点击在setting.json中编辑  然后找到terminal.integrated.profiles.windows开始编辑 一般来说要修改args这个地方***

```typescript
    "terminal.integrated.profiles.windows": {
        

        "PowerShell": {
            "source": "PowerShell",
            "icon": "terminal-powershell",
            "args": ["-NoExit","chcp 65001"]
        },
        "Command Prompt": {
            "path": [
                "${env:windir}\\Sysnative\\cmd.exe",
                "${env:windir}\\System32\\cmd.exe"
            ],
            "args": ["/K","chcp 65001"],
            "icon": "terminal-cmd"
        }
        }
    }
```
***然后就vscode终端就不会有乱码了***
***如果你希望打开powshell或cmd时候自动为utf-8编码。你可以这么做***
